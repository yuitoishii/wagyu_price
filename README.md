## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
自分のフォルダー内で管理してるか、ネット上で管理するか。


## プッシュとマージの違いは何でしょうか？
今いるブランチにセーブするか、セーブj内容を共有するか。


## コミットとプッシュの違い
変更を保存する。　変更内容をリモートに保存する。


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
変更内容を簡単に表記する。


## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
